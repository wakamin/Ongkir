Indonesia shipping cost and delivery tracking Wordpress plugin powered by RajaOngkir API.

== Description ==

## Features

- Shpping cost form
- Tracking form
- Support WooCommerce

## How to Use?

Basic usage

1. Input your RajaOngkir API key in Settings >> Ongkir >> Main Configuration
2. Select your RajaOngkir account type. Then click Save Changes button.
3. Click the Get Location Data, wait until finish.
4. In the Shipping Cost setting tab, you can set the available couriers to be displayed in the shipping cost form.
5. Tracking form is only for Basic and Pro account.
6. In the Tracking setting tab, you can set the available couriers to be displayed in the tracking form.
7. Use shortcode `[sdongkir]` in the post or page to display shipping cost and tracking form.
8. Shipping cost and tracking form also available as a widget called "Ongkir".

WooCommerce

1. Configure your store address in WooCommerce >> Settings >> General. For Pro RajaOngkir account, set your subdistrict address.
2. Go to Shipping WooCommerce settings tab, then click the Ongkir link at the top. On this page, you can toggle enable/disable shipping methods.


== Frequently Asked Questions ==

= What RajaOngkir account type that this plugin supports? =

This plugin support basic, starter, and pro RajaOngkir account type.
